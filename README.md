## Salary Survey Analysis using SQL, Excel, and Tableau

This project focuses on analyzing a salary survey dataset to uncover insights related to compensation, job roles, experience, education, gender, and industry trends.

## Project Overview

The analysis was carried out in three main stages:

Data Cleaning using Microsoft Excel

Data Analysis using SQL queries

Data Visualization using Tableau Dashboard

The goal of this project is to transform raw salary data into meaningful insights for better understanding workforce compensation patterns.

## Tools & Technologies

Microsoft Excel – Data Cleaning & Preprocessing

SQL – Data Analysis & Querying

Tableau – Data Visualization & Dashboard Creation

## Key Features

Standardized data types and handled missing values

Removed columns with excessive null values

Replaced inconsistent and invalid entries

Created new derived columns for analysis

Wrote SQL queries for:

Average and median salary analysis

Salary by industry, gender, and education

Compensation by job role and experience

Country-wise and city-wise salary trends

Additional monetary compensation analysis

##  Insights from Analysis

Average Salary: 90,893

Total Employees Analyzed: 24,830

United States has the highest total salary

Software Engineers form the largest workforce group

Employees with professional degrees earn higher salaries

Women participation is higher compared to men

Professionals with 40+ years of experience earn the most

## Visualization

An interactive Tableau dashboard was created to visualize:

Salary distribution by gender

Salary by experience and education

Country-wise salary mapping

Job title comparison

Key performance indicators
